# Portfolio Generator

## Description
A command-line tool that generates an HTML portfolio page from user input.

## Installation
This application utilizes the following node.js packages:

* `fs` for writing to the file system

* `inquirer` for collecting user input


## Usage
1. Clone this repository
2. Open Terminal (Mac) or Command Prompt (PC) and navigate to the folder where you cloned the repository
3. Run `node index.js` in the command-line. This will initiate the form
4. Provide your inputs
5. An `index.html` file will be created in the folder with your inputs
